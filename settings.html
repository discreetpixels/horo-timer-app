How to Use RubyTen Timer

Creating Timers

  • Task + Time: Type "design logo 25m" to create a 25-minute timer for design work
  • Time Only: Type "30m" for a 30-minute timer
  • Specific Time: Type "@3pm" to set timer until 3 PM
  • Quick Presets: Click preset buttons for common timers

Floating Timer Windows

  • Each timer gets its own floating window you can move around
  • Scrolling task name shows what you're working on
  • Control buttons: pause, play, stop, close
  • Click timer in main app to show floating window if hidden

Timer Completion

  • Celebration animation with particles (can be disabled)
  • Sound notification with custom volume control
  • "Run Again" button to restart the same timer
  • Timer history keeps track of recent timers

Menu Bar Integration

  • Shows countdown of next finishing timer
  • Click tray icon to open/hide main window
  • Right-click for context menu
